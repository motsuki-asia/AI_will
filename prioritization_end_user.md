## MoSCoW分類

### Must（MVP必須：これがないとサービスが成立しない）

| ID | 範囲 | 内容 |
|----|------|------|
| EU-001 | 登録・ログイン（メール） | メール+パスワードで登録/ログイン、利用規約同意、年齢確認への遷移 |
| EU-002 | 年齢確認・制限 | 年齢入力、年齢に応じたコンテンツ制限、未完了時の利用制限 |
| EU-003a | 一覧＋詳細 | Persona/Scenarioの一覧表示、詳細ページで説明・価格等の確認 |
| EU-005 | テキスト会話＋保存＋再開＋タイムアウトUX | Persona/Scenario選択→会話画面遷移、テキスト送信、スレッド保存・再開、応答遅延/タイムアウト時のエラー表示・再試行 |
| EU-007 | 通報（基本） | 通報開始、通報理由選択、受付完了表示 |
| EU-008 | 会話履歴削除 | 設定画面に削除導線、削除実行 |

### Should（重要：初期リリース後すぐに必要）

| ID | 範囲 | 内容 |
|----|------|------|
| EU-004 | 購入フロー | 購入開始→付与→二重購入防止、購入確認画面、失敗時の再試行、購入復元 |
| EU-006 | メモリー（基本） | メッセージ選択して保存、メモリー一覧、削除（確認あり） |
| EU-001 | 認証強化 | パスワードリセット導線、メール認証、登録/認証失敗時のエラー表示・再試行 |
| EU-003b | 検索・ステータス | キーワード検索、所持済み/未所持/無料のステータス表示 |
| EU-005 | 複数スレッド | 複数の会話スレッドを一覧で確認し、切り替え/再開 |
| EU-007 | ブロック・通報強化 | クリエイター/コンテンツ/タグ単位のブロック、ブロック対象の非表示、重複通報抑制、任意コメント入力 |
| EU-008 | 削除の詳細化 | Personaごと等の単位で削除、全データ削除時の猶予/注意事項 |

### Could（あると良い：リソース次第で後回し可能）

| ID | 範囲 | 内容 |
|----|------|------|
| EU-001 | ソーシャルログイン | Google/Apple/LINE等でのログイン |
| EU-003 | マーケット高度化 | フィルタ（性格タイプ、価格帯等）、ソート切替、サンプル（ボイス等） |
| EU-005 | 音声モード | 音声モード切替、マイク権限導線 |
| EU-006 | メモリー高度化 | タイトル/タグ/フォルダ付与、検索・並び替え、紐づき情報表示、ゴミ箱保持・復元 |
| EU-008 | エクスポート | データエクスポート導線、データ範囲明記、ステータス管理（受付/準備中/完了/失敗） |

## 実装Phase

### Phase 1: MVP（最小限の価値提供）

```
EU-001（登録・ログイン）
  ↓
EU-002（年齢確認・制限）
  ↓
EU-003a（一覧＋詳細）
  ↓
EU-005（テキスト会話＋保存＋再開＋タイムアウトUX）
  ↓
EU-007（通報：基本）
  ↓
EU-008（会話履歴削除）
```

**Phase 1完了で達成できること:**
- ユーザーが登録→年齢確認→Persona選択→会話を楽しめる
- 最低限の安全設計（年齢制限・通報・削除）でストア審査通過可能
- 無料コンテンツのみで価値検証可能

---

### Phase 2: 収益化・エンゲージメント強化

```
EU-004（購入フロー）
  ↓
EU-006（メモリー：保存/一覧/削除）
  ↓
EU-003b（検索/所持ステータス）
  ↓
EU-001（パスワードリセット/メール認証/エラー再試行）
  ↓
EU-007（ブロック/重複通報抑制/任意コメント）
  ↓
EU-005（複数スレッド切替）
```

**Phase 2完了で達成できること:**
- 有料コンテンツ販売による収益化
- メモリー機能でリテンション向上
- 検索・ブロックによる発見性・安全性向上
- パスワードリセット等でサポート負荷軽減

---

### Phase 3: UX高度化・差別化

```
EU-005（音声モード）
  ↓
EU-003（フィルタ/ソート/サンプル）
  ↓
EU-006（タグ/フォルダ/ゴミ箱復元）
  ↓
EU-008（エクスポート＋ステータス管理）
  ↓
EU-001（ソーシャルログイン）
```

**Phase 3完了で達成できること:**
- 音声会話による没入感・差別化
- マーケットプレイスの高度な探索体験
- メモリーの整理・復元でヘビーユーザー満足度向上
- プライバシー対応の完全化
