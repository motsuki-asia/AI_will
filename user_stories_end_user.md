# AI will - User Stories (End User)

> **Prioritization:** [prioritization_end_user.md](./prioritization_end_user.md)

---

## EU-001: アカウント登録

**As a** 新規ユーザー,  
**I want** メールアドレスまたはソーシャルログインでアカウントを作成したい,  
**So that** AI willのサービスを利用開始できる。

**受け入れ基準:**
- [ ] メールアドレス+パスワード、またはソーシャルログインで登録できる
- [ ] メール登録の場合、確認メールが送信され、リンククリックで認証完了となる
- [ ] 利用規約・プライバシーポリシーへの同意が完了しないと登録できない
- [ ] 登録完了後、年齢確認（または年齢帯選択）に進める
- [ ] 登録/認証に失敗した場合、理由が表示され再試行できる
- [ ] 既存ユーザー向けにメール/ソーシャルでのログイン導線がある
- [ ] メールログインではパスワードリセット導線があり、メール送付→リンクで再設定できる

---

## EU-002: 年齢確認と年齢に応じた制限

**As a** 登録済みユーザー,  
**I want** 年齢確認を完了したい,  
**So that** 年齢に応じた適切なコンテンツとサービスにアクセスできる。

**受け入れ基準:**
- [ ] 年齢確認の入力（生年月日または年齢帯）ができる
- [ ] 年齢に応じて閲覧可能コンテンツが制限される
- [ ] 未成年には課金上限などの制限が適用される（上限値は運用設定で変更可能）
- [ ] 年齢確認が未完了の場合、Persona/Scenarioの利用が制限される
- [ ] 年齢情報の変更は原則不可で、例外はサポート手続きとして案内される

---

## EU-003: Persona/Scenarioの検索・フィルタ・閲覧

**As a** ユーザー,  
**I want** マーケットプレイスでPersona/Scenarioを検索・フィルタして閲覧したい,  
**So that** 自分の好みに合ったAI彼女/彼氏やシチュエーションを見つけられる。

**受け入れ基準:**
- [ ] キーワード検索で名前・説明・タグに一致する結果が表示される
- [ ] フィルタ（性格タイプ、価格帯、レーティング、タグ等）で絞り込みできる
- [ ] ソート（人気/新着/評価/価格など）を切り替えられる
- [ ] 詳細ページで説明、価格、評価、レビュー件数、サンプル（テキスト/ボイス等）が確認できる
- [ ] 所持済み/未所持/無料などのステータスが分かる表示がある

---

## EU-004: Persona/Scenarioの購入と復元

**As a** ユーザー,  
**I want** Persona/Scenarioを購入し、必要に応じて購入状態を復元したい,  
**So that** いつでもコンテンツを利用できる。

**受け入れ基準:**
- [ ] 詳細ページから購入（または入手）を開始できる
- [ ] 購入前に価格・内容・支払い方法が確認できる
- [ ] 購入完了後、マイライブラリに追加され利用可能になる
- [ ] 購入失敗/キャンセル時に理由が表示され、再試行できる
- [ ] 購入の復元（Restore purchases）導線があり、購入状態が再同期できる
- [ ] 購入済みコンテンツは二重購入できない（所持済み表示で購入ボタン非活性等）
- [ ] 購入状態の不整合がある場合、復元操作でライブラリに反映される

---

## EU-005: 会話の開始・継続（テキスト/音声）

**As a** ユーザー,  
**I want** Persona/Scenarioを選んでテキスト/音声で会話を開始し継続したい,  
**So that** 疑似恋愛会話を没入して楽しめる。

**受け入れ基準:**
- [ ] Persona/Scenarioを選択して会話画面へ遷移できる（フリートークも可）
- [ ] テキスト送信で会話がスレッドに保存され、次回再開できる
- [ ] 音声モードへの切替ができ、マイク権限が未許可なら許可導線が表示される
- [ ] 応答遅延/タイムアウト時にエラー表示と再試行導線がある
- [ ] 不適切発言の通報導線が会話中に提供される
- [ ] 複数の会話スレッドを一覧で確認し、切り替え/再開できる

---

## EU-006: 私的メモリー（切り抜き）の保存と管理

**As a** ユーザー,  
**I want** 会話の一部を私的メモリーとして保存し整理したい,  
**So that** 後から見返して楽しめる。

**受け入れ基準:**
- [ ] 会話中の任意メッセージを選択してメモリーとして保存できる（複数選択可）
- [ ] 保存時にタイトル/タグ/フォルダ等のメタ情報を付与できる（任意）
- [ ] メモリー一覧で検索・並び替え・削除ができる
- [ ] 削除時は確認が表示され、削除後は一覧に表示されない
- [ ] Persona/Scenarioなど紐づく情報が確認できる
- [ ] 削除したメモリーは一定期間ゴミ箱に保持され復元できる（保持期間は仕様で定義）

---

## EU-007: 通報・ブロックによる安全な利用

**As a** ユーザー,  
**I want** 不適切コンテンツを通報し、必要ならブロックしたい,  
**So that** 安心して利用できる環境を保てる。

**受け入れ基準:**
- [ ] AI発言/Persona/Scenario/レビュー等から通報を開始できる
- [ ] 通報理由の選択と任意コメント入力ができる
- [ ] 通報受付完了が表示され、同一対象への重複通報が抑制される
- [ ] クリエイター/コンテンツ/タグ単位でブロックできる
- [ ] ブロック対象は検索/おすすめ/表示から抑制される（仕様範囲で定義）

---

## EU-008: プライバシー設定（削除・エクスポート）

**As a** ユーザー,  
**I want** 自分のデータを削除/エクスポートできる,  
**So that** プライバシーを自分でコントロールできる。

**受け入れ基準:**
- [ ] 設定画面に会話履歴の削除・データエクスポートの導線がある
- [ ] 会話履歴はPersonaごと等の単位で削除できる
- [ ] 全データ削除の場合、取り消し猶予/注意事項が明示される
- [ ] エクスポート要求後、ダウンロード導線が提供される（アプリ内 or メール等）
- [ ] エクスポートに含まれるデータ範囲（会話/メモリー/プロフィール等）が明記される
- [ ] エクスポート要求のステータス（受付/準備中/完了/失敗）を確認できる
